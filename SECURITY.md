# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.01.x  | :white_check_mark: |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

Astra AI wird aktiv mit Sicherheitsupdates für die aktuellen Hauptversionen (1.01.x, 1.0.x) versorgt. Ältere Versionen (< 1.0) erhalten keine Updates mehr. Dank integriertem Auto-Updater werden sicherheitsrelevante Patches automatisch verteilt, sobald sie verfügbar sind.

## Reporting a Vulnerability

Wenn du eine Sicherheitslücke in Astra AI findest, melde sie bitte vertraulich per E-Mail an:

voyeger95@gmail.com

Bitte gib eine möglichst genaue Beschreibung, ggf. Beispielcode und betroffene Versionen an. Wir bestätigen den Eingang innerhalb von 3 Werktagen und halten dich über den Fortschritt auf dem Laufenden. Kritische Lücken werden bevorzugt behandelt und schnellstmöglich gepatcht. Nach Abschluss der Analyse informieren wir dich über das Ergebnis und ggf. die Veröffentlichung eines Fixes.

Bitte veröffentliche keine Details zu Schwachstellen, bevor ein Fix bereitgestellt wurde (Coordinated Disclosure).
